## How to Play ğŸ•¹ï¸
1. Run the game using Python:
   ```bash
   python main.py
   ```
2. The board positions are numbered **1 to 9** like this:
   ```
   [1][2][3]
   [4][5][6]
   [7][8][9]
   ```
3. Enter the number of the cell where you want to place your **X**.
4. The computer will automatically place **O**.
5. The game continues until you win, lose, or draw.
